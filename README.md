# Call Center Analytics Dashboard

## Project Overview
This project is a **Call Center Analytics Dashboard** created using Excel to visualize key metrics of call center operations. It helps management monitor performance and improve customer satisfaction by analyzing call volumes, agent efficiency, resolution rates, talk duration, and customer satisfaction.

---

## Dashboard Visualizations
The Excel dashboard includes the following visuals:

1. **Count of Answered Calls** → KPI / List  
2. **Resolved Calls** → Pie Chart  
3. **Answered Calls (Yes/No)** → Line Chart  
4. **Average Talk Duration per Agent** → Clustered Column Chart  
5. **Average Satisfaction Rating per Agent** → Clustered Column Chart  
6. **Call Volume by Call ID** → Pie Chart showing count of calls  
7. **Satisfaction Rating Distribution** → Histogram  

These charts are combined into an **interactive Excel dashboard** for easy analysis of call center performance.

## Project Objective
- Monitor call center performance at a glance  
- Analyze agent efficiency and customer satisfaction  
- Identify trends, bottlenecks, and areas for improvement
